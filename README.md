Calculadora Básica en Python

Este proyecto consiste en una calculadora básica desarrollada en Python utilizando un módulo personalizado.

Descripción

El programa permite al usuario realizar operaciones matemáticas básicas:

- Suma
- Resta
- Multiplicación
- División (con validación para evitar división entre cero)

El proyecto está estructurado en dos archivos principales:

- Operaciones.py → Contiene las funciones matemáticas
- main.py → Contiene la interacción con el usuario

---

Funcionalidades

- Solicita dos números al usuario
- Permite seleccionar una operación mediante un menú
- Ejecuta la operación usando funciones de un módulo
- Muestra el resultado en pantalla
- Permite repetir operaciones
- Maneja errores en la entrada de datos
